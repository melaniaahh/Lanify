# Lanify
Lanify is Web App for Managing Spotify Playlist

## Live Website
Open https://lanify.vercel.app/

## Local Env Installation
Repo Clone & Change Directory
```bash
git clone https://github.com/melaniaahh/lanify.git

cd lanify

```
Install Required Dependency
```bash
npm install
```

Then, Run
```bash
npm start
```


## Feature
Login and Playlist
